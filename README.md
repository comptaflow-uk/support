# ComptaFlow UK by MeeTempo

⚠️ Bookkeeping assistance tool - Does not replace a qualified accountant

## Description

WordPress plugin to simplify bookkeeping and data organisation for UK sole traders, freelancers & small businesses.

## Installation

1. Download the .zip file
2. WordPress Admin → Plugins → Add New → Upload Plugin
3. Activate the plugin
4. Enter your licence key

## Features

* Income & expense management
* Guided bookkeeping with UK nominal codes (Sage/Xero compatible)
* CSV Exports: Trial Balance, General Ledger, Journal, Profit & Loss
* Bank reconciliation
* UK VAT management - MTD ready (Pro plans)
* Self Assessment & Corporation Tax reports
